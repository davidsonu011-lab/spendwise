 # SpendWise

SpendWise is a simple and smart expense tracking app made for people who want to manage daily money without confusion.

The idea behind this app is very simple:
money should feel easy to track.

Most finance apps look complicated, overloaded, or stressful.
SpendWise was created with a cleaner and calmer approach so users can quickly add expenses, check spending habits, and stay organized without learning anything difficult.

The app focuses on speed, simplicity, and clarity.

Users can:
- add daily expenses
- track income and spending
- check total balance
- organize financial records
- view clean summaries
- manage personal budgets easily

Everything is designed with a minimal and modern interface so the experience feels smooth and distraction-free.

SpendWise is built using modern Android development tools including:
- Kotlin
- Jetpack Compose
- Material Design

The app is lightweight, responsive, and designed for everyday use.

Whether someone is a student, freelancer, shop owner, or working professional, SpendWise helps keep financial activity organized in a simple way.

This project was created with passion for clean UI, practical tools, and real user experience.
